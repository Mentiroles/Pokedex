Instrucciones

"¡Mirá vó', te presento mi proyecto de Pokémon! En esta movida que armé, creé una página web que te permite buscar información sobre los Pokémon más picantes. En la página principal, armé un quilombo con un formulario para que puedas poner el nombre de un Pokémon. Cuando le des al botoncito de buscar, mi proyecto se pone las pilas y hace una consulta a la API de Pokémon para traer los datos del bicho que estás buscando. Con esos datos, armé una ficha re groso con la info clave, como el nombre, la altura, el peso y las estadísticas de combate del Pokémon. Además, le di un toque personalizado a la ficha, resaltando el tipo del Pokémon con unos colores copados que representan cada tipo. Para hacer todo esto, me mandé con el HTML para darle forma a la página, el CSS para ponerle estilo a los elementos y el JavaScript para hacer las movidas con la API y manipular los datos. También me agarré un archivo JSON con información sobre los Pokémon para hacer más fácil la búsqueda. Mi proyecto no es puro cháchara, che. Tiene una interfaz de primera, con colores y estilos que le dan el toque bien Rosarino. "
